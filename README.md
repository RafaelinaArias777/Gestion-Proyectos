# 🚀 Gestión-Proyectos

Una aplicación robusta diseñada para facilitar la planificación, organización de tareas y el monitoreo en tiempo real del progreso de proyectos individuales o colaborativos.

## 📋 Características Principales

* **Creación de Proyectos:** Define objetivos, fechas límite y presupuestos.
* **Tablero de Tareas:** Organización visual mediante estados (To-Do, In Progress, Done).
* **Asignación de Roles:** Gestiona los miembros del equipo y sus responsabilidades.
* **Panel de Control (Dashboard):** Gráficos interactivos para medir el avance del proyecto.
* **Notificaciones:** Alertas sobre fechas de entrega próximas y actualizaciones de tareas.
